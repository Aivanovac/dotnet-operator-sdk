GetCurrentNamespace(String)

Returns the name of the current namespace. To determine the current namespace the following places (in the given order) are checked:

  1. The created kubernetes configuration (from file / incluster)
  2. The env variable given as the param to the function (default "POD_NAMESPACE") which can be provided by the kubernetes downward API
  3. The fallback secret file if running on the cluster (/var/run/secrets/kubernetes.io/serviceaccount/namespace)
  4. "default"
